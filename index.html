Skyflow.io

Connect Enterprise Clouds seemlessly

Download this project as a .zip file Download this project as a tar.gz file

Enterprise-level companies use more and more Cloud services from multiple parties (Salesforce, SAP, Microsoft, IBM, HP, Oracle…) & API from many SAAS providers. IoT and customer facing apps generate even more data, events and information flows.

  • Classical integration systems don’t fit their needs anymore
  • The connections between external events (web, forms, partners, clients, IoT) and internal “business” clouds are not structured

Skyflow.io is the #1 “orchestra conductor” for all Enterprise Clouds

Knowledge

One company’s knowledge can be described in terms of

  • Data (CRM, ERP, Marketing database, HR, custom business apps…)
  • Links / flows / integration >> Here comes Skyflow.io

From external events to Enterprise value

  • Enterprise-level companies have dozens of websites, forms, apps, and IoT is adding a new field of services and associated data.
  • Skyflow lets centralize the flows, providing a secured - but unauthenticated - entry point to trigger processes and route data to specific Enterprise Clouds.

Flexibility for Cloud integrations

  • Enterprise Clouds need to communicate
  • But not replicate all data
  • Integration policies are more and more split // knowledge is dispatched
  • Skyflow lets unify a central - straightforward way to integrate these Clouds
  • Skyflow is “process oriented” (not “data oriented”). Means we focus on the links, not on the destinations. Skyflow can be used as a compute engine for these clouds.
  • These links represent an important part of the knowledge, Skyflow lets one company centralize the associated rules & documentations.

Trusted Platform

  • OpenSource core platform & framework
  • Ecosystem of free or licences addons
  • Ecosystem of professional services partners:
  • Cloud System Integrators
  • Digital Agencies
  • Enterprise support
  • 1 click deploy to Heroku platform (by salesforce.com)

Built for both Developers & Business

  • Developers: get the flexibility Enterprise Clouds integration needs. They focus on the links, Skyflow to provide the framework for authentications, security, a simple API (external events) and a large library of Enterprise Clouds methods.
  • Business: has a global vision of all the flows from an understandable web interface where they find events to flows mapping, flows documentations and rules, and provisionned addons.
