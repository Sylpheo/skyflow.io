Skyflow.io is the maestro for all your clouds

Skyflow.io is an ETL framework that runs on top of Heroku. It seamlessly connects enterprise data and external events to business clouds from multiple providers (Salesforce, Wave, ExactTarget and more APIs). It features an interface to manage flows, a complete PHP library to script them, and a simple API for IoT.
